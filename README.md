# Blackboard

![license](https://img.shields.io/badge/license-MIT-blue)![version](https://img.shields.io/badge/version-1.0-blue)



JSP를 이용한 자유게시판



## 목적

* JSP를 이용하여 커뮤니티 자유게시판을 제작했다.
* Maria DB를 사용하여 입력한 값들을 저장했다.
* 웹페이지에서 직접 게시글을 작성하여 DB에 저장하도록 한다.



## 사용한 프레임워크

* Maria DB 10.5.8
* Inteli J
* JSP
